Support

Comment pouvons-nous aider ?

Pour toute aide sur AGBoard - PCBEditor, demande de fonctionnalité, signalement de bug, problème d'import de fichier ou question App Store, contactez le support.

Contact

E-mail : support@agboard-app.com

Merci d'indiquer votre modèle d'iPhone ou d'iPad, la version d'iOS, la version de l'app, le format de fichier ouvert et une courte description du problème ou de l'amélioration souhaitée.

Avant d'écrire

Avertissements d'import

Ouvrez Analysis → Import report dans l'app et joignez le texte de l'avertissement à votre e-mail.

Signaler un bug

Utilisez Help → Report bug. L'app prépare un message de diagnostic que vous pouvez relire avant envoi.

Demander une fonctionnalité

Utilisez Help → Request feature pour demander une nouvelle fonctionnalité, une amélioration, une extension de workflow ou un format de fichier. Nous étudions rapidement les demandes.

Fichiers composants

Si un import CSV/TXT échoue, notez le délimiteur, les unités, les lignes ignorées et le profil d'analyse sélectionné.

Fichiers pris en charge

AGBoard - PCBEditor prend en charge les Gerber RS-274X/X2, les perçages Excellon, les jobs ODB++, les cartes Eagle, les fichiers job Gerber (.gbrjob), les netlists IPC-D-356 et les placements de composants CSV/TXT/POS.
