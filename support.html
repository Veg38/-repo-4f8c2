Support

How can we help?

For help with AGBoard - PCBEditor, bug reports, feature requests, file import issues or App Store questions, contact support.

Contact

Email: support@agboard-app.com

Please include your iPhone or iPad model, iOS version, app version, the file format you opened, and a short description of what happened or what you would like to improve.

Before you email

Import warnings

Open Analysis -> Import report in the app and include the warning text in your email.

Bug report

Use Help -> Report bug. The app prepares a diagnostic message that you can review before sending.

Feature request

Use Help -> Request feature to ask for a new feature, workflow improvement, extension or file format. We review requests quickly.

Component files

If CSV/TXT import fails, note the delimiter, units, skipped rows and selected parser profile.

Supported files

AGBoard - PCBEditor supports Gerber RS-274X/X2, Excellon drill files, ODB++ jobs, Eagle board files and CSV/TXT component placement files.
